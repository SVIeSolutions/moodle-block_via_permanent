The Via Permanent Bloc can only be used if you have the Via activity module.
This bloc creates shortcuts to the Via permanent activities* that have been shared within the course.

To Install
1. Copy the progress directory to the blocks directory of your Moodle instance
2. Visit the notifications page

For more information, visit...
http://docs.moodle.org/en/Installing_contributed_modules_or_plugins

Once the bloc is installed, you can use it in a course as follows.

1. Turn editing on
2. Add the Via Permanent block to your page (http://docs.moodle.org/en/Blocks) where you wish it to appear.
3. Create your Via activities as normal
4. Any permanent activity will appear atomatically in the list.

*Permanent activities have no begining or end date,
you may control its' access with the waiting room. See the Via guide for reference.
